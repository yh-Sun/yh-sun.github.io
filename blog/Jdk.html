# JDK部署

https://blog.csdn.net/love3765/article/details/88783126

https://blog.csdn.net/qq_37345604/article/details/80556639

两种安装模式，一种是npm安装、一种是纯离线tar解压+配置环境变量安装：

# npm安装:

D:\WorkBook\华通互惠\保司2相关文档\服务器资源\总结\jdk\

# 上传安装包

scp jdk-8u211-linux-x64.rpm kscd@10.1.4.193:~/deploy/jdk

# 安装jdk

rpm -ivh jdk-8u211-linux-x64.rpm

# 配置环境变量(rpm不用配置环境变量)

export JAVA_HOME=/home/kscd/jdk/jdk1.8.0_251 export PATH=$PATH:$JAVA_HOME/bin export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

# tar离线安装:

# 上传安装包

scp jdk-8u251-linux-x64.tar.gz kscd@10.1.4.193:~/deploy/jdk

# 解压JDK压缩包,并修改目录名称

[root@localhost jdk]# tar -zxvf jdk-8u251-linux-x64.tar.gz
[root@localhost jdk]# mv jdk1.8.0_251  jdk1.8

# 配置该用户下的环境变量

vi ~/.bash_profile
source ~/.bash_profile

以下为命令实例，供参考：

scp /home/kscd/deploy/jdk/jdk-8u251-linux-x64.tar.gz kscd@10.1.1.100:/home/kscd/deploy/jdk

tar -zxvf jdk-8u251-linux-x64.tar.gz


cp -r jdk1.8.0_251/ /home/kscd/
mv jdk1.8.0_251 jdk1.8

vi ~/.bash_profile

JAVAHOME=/home/kscd/jdk1.8/bin
PATH=$PATH:$HOME/.local/bin:$HOME/bin:$JAVAHOME

source ~/.bash_profile
