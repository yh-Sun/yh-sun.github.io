# 安装步骤

1.上传包到/home/kscd/elk/目录下

scp elasticsearch-7.6.0-linux-x86_64.tar.gz kscd@10.1.10.100:/home/kscd/deploy/elk
scp kibana-7.6.0-linux-x86_64.tar.gz kscd@10.1.10.100:/home/kscd/deploy/elk
scp elasticsearch-head.tar.gz kscd@10.1.10.100:/home/kscd/deploy/elk
scp node-v12.16.1-linux-x64.tar.xz kscd@10.1.10.100:/home/kscd/deploy/elk

2.解压

3.安装elasticsearch

1.进入elasticsearch目录下，修改config下的elasticsearch.yml
cd elasticsearch-7.6.0/config
vi elasticsearch.yml

2.具体修改项，如下：
cluster.name: kscd
node.name: node-1
path.data: /home/kscd/es/data
path.logs: /home/kscd/es/logs
network.host: 10.1.10.100
http.port: 9200
http.cors.enabled: true
http.cors.allow-origin: "*"
discovery.seed_hosts: ["10.1.10.100", "[::1]"]
cluster.initial_master_nodes: ["node-1"]

3.需要root修改几处配置文件
详情：https://www.jianshu.com/p/2285f1f8ec21
vi /etc/security/limits.conf
    * soft nofile 65536
    * hard nofile 65536
    * soft nproc 4096
    * hard nproc 4096

vi /etc/sysctl.conf
	vm.max_map_count=262144
sysctl -p


4.进入bin目录下启动elasticsearch
	正常启动：./elasticsearch
	后台启动：./elasticsearch -d -p pid

5.访问：http://10.1.10.100:9200/

4.安装kibana

1.进入kibana配置文件config目录
cd /home/kscd/deploy/elk/kibana-7.6.0-linux-x86_64/config
vi kibana.yml

2.具体修改项，如下：
server.port: 5601
server.host: "10.1.10.100"
elasticsearch.hosts: ["http://10.1.10.100:9200"]

3.进入bin目录下启动kibana
	正常启动：./kibana
	后台启动：nohup ./kibana &

4.访问：http://10.1.10.100:5601/

5.安装node+head

1.es-head需要安装node，所以需要解压node，并配置环境变量
vi ~/.bash_profile
	NODEHOME=/home/kscd/deploy/elk/node-v12.16.1-linux-x64/bin
source ~/.bash_profile

2.查看node是否安装成功
    [kscd@localhost ~]$ node -v
    v12.16.1
    [kscd@localhost ~]$ npm -v
    6.13.4

3.安装head
cd /home/kscd/deploy/elk/elasticsearch-head
cat package.json
看到是start，所以执行：npm run start安装

4.vi Gruntfile.js
connect: {
    server: {
        options: {
            port: 9110,
            base: '.',
            keepalive: true
        }
    }
}
修改为：
connect: {
    server: {
        options: {
        	hostname: '0.0.0.0',
            port: 9110,
            base: '.',
            keepalive: true
        }
    }
}

5.重新启动一下，后台启动
nohup npm run start &

6.完成
http://10.1.10.100:9110/
	搜索框输入：http://10.1.10.100:9200/
